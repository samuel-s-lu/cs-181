Problem 2

One way to represent directed graphs is with an adjacency list of sorts. First label all vertices from 1 to nnn. Recall that it will take log⁡n\log{n}logn bits to represent integers up to nnn, and a total of

2log⁡n+22\log{n}+22logn+2

bits to make the encoding prefix-free. Our representation will be a list of lists, where the iii-th list enumerates the neighbors of the iii-th vertex. So for each vertex, it will take up to

10⋅(2log⁡n+2)=20log⁡n+2010\cdot (2\log{n}+2)=20\log{n}+2010⋅(2logn+2)=20logn+20

bits to represent its neighbors (since number of neighbors capped at 10). We need to again convert this list encoding into one that is prefix-free, which will take

2⋅(20log⁡n+20)+2=40log⁡n+422\cdot (20\log{n}+20)+2=40\log{n}+422⋅(20logn+20)+2=40logn+42

n⋅(40log⁡n+42)=40nlog⁡n+42nn\cdot(40\log{n}+42)=40n\log{n}+42nn⋅(40logn+42)=40nlogn+42n

This is a valid encoding because it just uses compositions of integer encodings, conversion to prefix-free encodings, and concatenations of prefix-free encodings, which were all shown in lecture to be valid encoding functions with a respective decoding function.
